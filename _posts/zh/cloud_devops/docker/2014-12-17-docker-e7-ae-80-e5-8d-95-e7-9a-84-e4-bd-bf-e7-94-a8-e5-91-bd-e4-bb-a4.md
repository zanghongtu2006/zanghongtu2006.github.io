# Hello World

使用`[docker run]命令在docker container中运行应用程序`
[codesyntax lang="bash"]

```
# docker run fedora /bin/echo 'hello world'
hello world
```

[/codesyntax]
`docker run fedora`，使用fedora的image，启动一个container
`/bin/echo 'hello world'`，fedora中可执行的命令，输出“hello world"

# 交互式container

[codesyntax lang="bash"]

```
# docker run -t -i fedora /bin/bash
bash-4.3#
```

[/codesyntax]
`-t` 在container中，分配一个terminal
`-i` 创建一个交互式连接
交互执行完成后，使用exit或Ctrl+D退出

# Daemonized Docker

[codesyntax lang="bash"]

```
# docker run -d fedora /bin/sh -c "while true; do echo hello world; sleep 1; done"
7f22b335fb2cb2342475f720eaaf666703c97f96ea07b8a1b23e020c21a9f1a7
```

[/codesyntax]
`-d 创建一个daemon，并在后台执行命令`
`7f22b335fb2cb2342475f720eaaf666703c97f96ea07b8a1b23e020c21a9f1a7 container ID`
可以使用`docker ps查看docker daemon信息`
[codesyntax lang="bash"]

```
# docker ps
CONTAINER ID        IMAGE               COMMAND                CREATED             STATUS              PORTS               NAMES
7f22b335fb2c        fedora:latest       "/bin/sh -c 'while t   5 minutes ago       Up 5 minutes                            silly_archimedes
```

[/codesyntax]
daemon信息中，有docker的命令，ID，状态和名字，名字是自动生成的。
通过`docker logs`命令，查看daemon的标准化输出
[codesyntax lang="bash"]

```
# docker logs silly_archimedes
hello world
hello world
hello world
。。。
```

[/codesyntax]
通过`docker stop`命令，停止指定的docker daemon
[codesyntax lang="bash"]

```
# docker stop silly_archimedes
silly_archimedes
```

[/codesyntax]
再次查看docker daemon
[codesyntax lang="bash"]

```
# docker ps
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES
```

[/codesyntax]
已经没有之前的docker daemon了
